\section{Installation}
Une fois le fichier de l'application décompressé ({\policecommande tar -zxvf
quadricolor.tgz}), il suffit de taper la commande {\policecommande
make}.
%PRECISIONS SUR QT

\section{Exécution}
Pour exécuter l'application, il suffit de lancer l'exécutable
{\policecommande quadricolor}.

Par la suite, vous pouvez choisir une image en cliquant sur
File$\rightarrow$Open. Le nom de l'image s'affichera alors dans la liste de
gauche. La liste des images contenues dans le cache est affichée dans
le terminal.

En fait, lorsque l'image est dans la liste, elle n'est pas
nécessairement affichée comme étant dans le cache. Pour cette raison,
nous vous recommandons de double cliquer sur l'image que vous venez
d'ajouter.
